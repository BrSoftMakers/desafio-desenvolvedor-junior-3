O blog foi desenvolvido utilizando as seguintes tecnologias:
•	Node.js
•	Express
•	TypeScript
•	Next.js
•	ORM Prisma

Para que funcione a aplicação é necessário:
1.	Instalar o banco de dados PostgreSQL;
2.	Criar o banco de dados blog;
3.	Criar as tabelas através das migrations presentes no backend;

As duas aplicações ‘backend’ e ‘frontend’ devem estar rodando simultaneamente através do comando:
•	npm run dev
